# 📚 Book Scraper

A Python web scraper that extracts book titles and prices from [books.toscrape.com](https://books.toscrape.com) across all 50 pages and saves the data into a CSV file.

## 🛠️ Libraries Used
- `requests`
- `BeautifulSoup4`
- `csv`

## ⚙️ How to Run
1. Install dependencies:
   pip install requests beautifulsoup4
2. Run the script:
   python scraper.py
3. Output will be saved as `books.csv`

## 📊 Output
A CSV file containing 1000 books with:
- Book Title
- Price
